## Key Design Principles at a Glance

```
1. SECURITY-FIRST AESTHETIC
   Dark theme, neon accents, professional tone
   
2. CLARITY OVER CLEVERNESS
   Direct information, no hidden complexity
   
3. ACCESSIBILITY BY DESIGN
   High contrast, multiple channels, WCAG AA
   
4. USER EDUCATION FIRST
   Every element teaches phishing awareness
   
5. CONSISTENCY ACROSS INTERFACES
   Same logic, same emoji, same colors
   
6. PROGRESSIVE DISCLOSURE
   Show info as needed, not all at once
```
