# Quick Start - Enhanced GUI Features

## 🎯 Fastest Way to Use

### Method 1: Copy & Paste Email (Easiest)
```
1. Copy the suspicious email
2. Select "🤖 Auto-Detect (Recommended)"
3. Paste in text area (Ctrl+V)
4. Click "🔍 ANALYZE"
5. View results
```

### Method 2: Copy & Paste URL
```
1. Copy the suspicious URL
2. Select "🤖 Auto-Detect (Recommended)" or "🔗 URL Only"
3. Paste in text area (Ctrl+V)
4. Click "🔍 ANALYZE"
5. View results
```

### Method 3: Paste Email Address
```
1. Copy the sender's email (e.g., from email header)
2. Select "🤖 Auto-Detect" or "📧 Email Only"
3. Paste in text area (Ctrl+V)
4. Click "🔍 ANALYZE"
5. View results
```

## ✅ What Works

| Input Type | Example | Mode |
|-----------|---------|------|
| Single Email | user@domain.com | Any |
| Single URL | https://domain.com | Any |
| Full Email | From: sender@... | Auto-Detect |
| Multiple Emails | addr1@... addr2@... | Auto-Detect |
| Mixed Content | Email + URL + Text | Auto-Detect |
